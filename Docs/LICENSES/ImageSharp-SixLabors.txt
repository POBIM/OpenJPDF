Library: SixLabors.ImageSharp
Purpose: Image processing and manipulation library
Version: 3.1.5
Website: https://sixlabors.com/
Repository: https://github.com/SixLabors/ImageSharp
Documentation: https://docs.sixlabors.com/articles/imagesharp/

USAGE:
This library provides advanced image processing capabilities for OpenJPDF including:
- Image loading, processing, and manipulation
- Format conversion (PNG, JPEG, WEBP, etc.)
- Image filtering and transformations
- Pixel-level image operations
- Image composition and layering

FEATURES:
- Pure .NET implementation (no external dependencies)
- High-performance image processing
- Support for multiple image formats
- Thread-safe operations
- GPU acceleration support
